### Quick Start
1. Enter playground 
```sh 
cd ./playground
```

2. Launch the dev Environment
```sh
npm run dev
```

### Footage
* To run the dev environment
```sh
cd ./src/node/server
# Add to devDependencies
pnpm add <package> --dev 
```
